import numpy as np
import scipy.stats as sps

rng = np.random.default_rng()

n = 12
x = rng.normal(10, 2, size=n)
y = rng.normal(10, 2, size=n)

r = np.corrcoef(x, y)[0, 1]
r

0.1192933209112766

x_bar = np.mean(x)
y_bar = np.mean(y)
x_bar, y_bar

(10.388913240287662, 11.064291649931336)

Write the deviations as e.g. \(d_{x_i}\), so:

\[\begin{split} d_{x_i} = x_i - \bar{x} \\ d_{y_i} = y_i - \bar{y} \end{split}\]
d_x = x - x_bar
d_y = y - y_bar

As before, the z-scores are:

\[\begin{split} z_{x_i} = \frac{d_{x_i}}{\sigma_x} \\ z_{y_i} = \frac{d_{y_i}}{\sigma_y} \end{split}\]
s_x = np.sqrt(np.mean(d_x ** 2))
s_y = np.sqrt(np.mean(d_y ** 2))
s_x, s_y

(1.7110483354317079, 2.16088917390988)

np.std(x), np.std(y)

(1.7110483354317079, 2.16088917390988)

z_x = d_x / s_x
z_y = d_y / s_y

# r (correlation) again.
r_again = np.mean(z_x * z_y)
np.isclose(r, r_again)

True

fit = sps.linregress(x, y)
fit

LinregressResult(slope=0.15065596940713855, intercept=9.499139854629142, rvalue=0.1192933209112766, pvalue=0.7119261473760863, stderr=0.39651335249327335, intercept_stderr=4.174839448061959)

# The slope, recalculated from r
b = r * s_y / s_x
np.isclose(fit.slope, b)

True

Remember, we can reconstruct the best-fit intercept from the best-fit slope with:

\[ c = \bar{y} - \bar{x} b \]
# Intercept, recalculated from r
c = y_bar - x_bar * b
np.isclose(fit.intercept, c)

True

Next, rewrite slope \(b\) in terms of \(d_{x_i}, d_{y_i}\):

\[\begin{split} b = \frac{r \sigma_y}{\sigma_x} \\ = \frac{\frac{1}{n}\Sigma ( z_{x_i} z_{y_i} ) \sigma_y}{\sigma_x} \\ = \frac{\frac{1}{n \sigma_y \sigma_x }\Sigma ( d_{x_i} d_{y_i} ) \sigma_y}{\sigma_x} \\ = \frac{1}{n \sigma_x^2 }\Sigma ( d_{x_i} d_{y_i} ) \\ = \frac{1}{\Sigma (d_x)^2 }\Sigma ( d_{x_i} d_{y_i} ) \end{split}\]

Therefore:

\[\begin{split} b = \frac{1}{\Sigma (d_x)^2 }\Sigma ( d_{x_i} d_{y_i} ) \\ = \frac{1}{\Sigma (d_x)^2 }\Sigma ( d_{x_i} (y_i - \bar{y}) ) \\ = \frac{1}{\Sigma (d_x)^2 }(\Sigma ( d_{x_i} y_i ) - \bar{y} \Sigma d_{x_i}) )) \\ = \frac{1}{\Sigma (d_x)^2 }\Sigma d_{x_i} y_i \end{split}\]

Really though?

b_again = np.sum(d_x * y) / np.sum(d_x ** 2)
np.isclose(b, b_again)

True

Finally, rewrite the slope in terms of \(y_i\) and \(x_i\):

\[\begin{split} b = \frac{1}{\Sigma (d_x)^2 }\Sigma d_{x_i} y_i \\ = \frac{1}{\Sigma (x_i - \bar{x})^2 } \Sigma (x_i - \bar{x}) y_i \\ = \frac{1}{\Sigma x_i^2 - 2 \bar{x} \Sigma x_i + n \bar{x}^2 } ( \Sigma x_i y_i - \bar{x} \Sigma y_i ) \\ = \frac{1}{\Sigma x_i^2 - n \bar{x}^2 } ( \Sigma x_i y_i - n \bar{x} \bar{y} ) \\ \end{split}\]

Really?

b_yet_again = ((np.sum(y * x) - n * x_bar * y_bar) /
               (np.sum(x ** 2) - n * x_bar ** 2))
np.isclose(b, b_yet_again)

True

Then consider the general linear model.

Compile the design \(X\) from \(\vec{d_x}\) and a column of ones.

Now:

\[\begin{split} X^T\vec{y} = \begin{bmatrix} \Sigma (d_{x_i} y_i) \\ n \bar{y} \end{bmatrix} \\ X^T X = \begin{bmatrix} \Sigma (d_{x_i})^2 && \Sigma d_{x_i} \\ \Sigma d_{x_i} && n \end{bmatrix} \\ = \begin{bmatrix} \Sigma (d_{x_i})^2 && 0 \\ 0 && n \end{bmatrix} \\ (X^T X)^{-1} = \begin{bmatrix} \frac{1}{\Sigma (d_{x_i})^2} && 0 \\ 0 && \frac{1}{n} \end{bmatrix} \\ (X^T X)^{-1} X^T\vec{y} = \begin{bmatrix} \frac{1}{\Sigma (d_{x_i})^2} \Sigma (d_{x_i} y_i) \\ \bar{y} \end{bmatrix} \end{split}\]

You will recognize the first term as the slope we calculated above. The second is the intercept for the best-fit line of \(d_x\) and \(y\). Call this intercept \(c_{d_x}\). We can check what this value should be by using the formula above for getting the intercept given the slope.

\[\begin{split} c_{d_x} = \bar{y} - \bar{d_x} b \\ = \bar{y} \end{split}\]

The general linear model has found the correct slope and intercept, as we derived them above from the correlation \(r\).
